Tentang Saya

Saya memiliki latar belakang pendidikan di bidang Teknik Komputer dan Jaringan (TKJ) , yang membentuk pemahaman saya tentang bagaimana sistem, jaringan, dan data saling terhubung.

Seiring waktu, fokus saya berkembang ke Web Development , khususnya pada backend dan logika sistem, hingga akhirnya memperdalam minat di bidang Web Security dan Cyber Security.

TKJ → Dasar jaringan & sistem
Web Development → Backend & arsitektur
Cyber Security → Secure-by-design mindset

Saat ini, saya berfokus pada pengembangan aplikasi web dengan pendekatan security-first , terutama pada sisi backend, autentikasi, dan alur data.

Saya percaya bahwa sistem yang baik bukan hanya berjalan, tetapi juga aman, dapat dipelihara, dan tahan terhadap kesalahan desain.

02

Menganalisis Alur Data

03

Mempertimbangkan Keamanan

Bagi saya, kode bukan hanya tentang “berjalan atau tidak”, tetapi tentang bagaimana sistem dapat bertahan, berkembang, dan tetap aman seiring waktu.

Security Mindset

Security Awareness & Defense Principles

Saya membangun sistem dengan mindset pertahanan, bukan sekadar menambal celah setelah terjadi masalah.

Think in Attack Surface

Setiap endpoint, input, dan data flow saya anggap sebagai potensi titik serang yang perlu dikontrol sejak desain awal.

Secure by Default

Konfigurasi awal sistem harus aman tanpa bergantung pada asumsi bahwa user atau environment selalu ideal.

Assume Breach

Sistem saya rancang dengan asumsi kebocoran bisa terjadi, sehingga dampak dan akses harus tetap terbatas.
